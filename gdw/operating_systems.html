Computer Operating Systems

Basic Operating System Concepts

Operating Systems
-The software that supports a computer's basic functions, such as scheduling tasks, executing applications, and controlling peripherals.
Batch Processing
-The processing of previously collected jobs in a single batch.
Timesharing
-The operation of a computer system by several users for different operations at the same time.
Multitasking
-Being able to execute more than one program or task simultaneously.
Multiprocessing
-The running of two or more programs or sequences of instructions simultaneously by a computer with more than one central processor.

System Boot
-When a computer loads up large programs onto its RAM
CISC vs RISC
-CISC (Complex Instruction Set Computer) is a computer in which single instructions can execute several low-level operations or are capable of multi-step operations or addresssing modes with single instructions.
-RISC (Reduced Instruction Set Computer) is a computer based processor or processors designed to perform a limited set of operations extremely quickly.
